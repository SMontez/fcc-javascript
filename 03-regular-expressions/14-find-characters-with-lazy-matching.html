Original Code

let text = "<h1>Winter is coming</h1>";
let myRegex = /<.*>/;
let result = text.match(myRegex);

New Code - from FCC

let text = "<h1>Winter is coming</h1>";

let myRegex = /<h1>?/; <-- This works but can be done with a string literal and w/o the lazy (?) match

let result = text.match(myRegex);

New Code - Mine

let text = "<h1>Winter is coming</h1>";
let myRegex = /<[h1]*>?/;
let result = text.match(myRegex);

I prefer THIS - It does work w/o the lazy (?) match, but won't get full string "<h1>" w/o the *
